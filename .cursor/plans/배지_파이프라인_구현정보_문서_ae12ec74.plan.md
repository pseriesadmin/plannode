## 문서에 넣을 섹션 (정밀도 기준)

1. **메타**  
   - 대상 커밋/날짜(선택), “원본 플랜 경로(개요 1문장)”, 문서 범위(구현 / 비구현).

2. **요약 표: 플랜 대비**  
   - 구현됨: `src/lib/ai/*`, `Node.metadata`, 파일럿 3트랙, PRD, 트리 export, vitest, `pilotBridge` `metadata` diff, [`NEXT7_SCOPE.md`](.cursor/harness/NEXT7_SCOPE.md)에 적힌 후속.  
   - **의도적 비구현(플랜 §7과 동일)**: `plan_nodes`·RLS, 서버 `callAI`, v3 **전부** `PROMPT_MATRIX`.

3. **데이터 모델 (정확한 단일 흐름)**  
   - `BadgeSet` / `Node.metadata` (참고: [`src/lib/ai/types.ts`](src/lib/ai/types.ts), [`src/lib/supabase/client.ts`](src/lib/supabase/client.ts)).  
   - 읽기: [`getBadgeSetFromNodeInput`](src/lib/ai/badgePromptInjector.ts) (레거시 `badges` → `migrateLegacyBadgesToSet` 경로 한 문단).  
   - 쓰기: 파일럿 `applyBadgeSetToNode` + `badges` 평면 동기화(플랜 §2와 동일 취지).

4. **모듈 지도 (파일·책임)**  
   - `badgePromptInjector`: `buildBadgeContext`, `formatBadgeTracksForDisplay`, `BADGE_PROMPT_FRAGMENTS`, `shouldForceSonnet` (**라우터 미연결** 명시).  
   - `promptMatrix` / `getSystemPrompt`: `OutputIntent`·`PROMPT_MATRIX.root` 범위.  
   - `iaExporter`: `nodesToTreeText`, `buildPrompt`(병합 `BadgeSet` + 트리 + `getSystemPrompt`), `formatPromptForClipboard`, `generateDocumentFromPrompt` **스텁**.  
   - `prdStandardV20`: `getBadgeSetFromNodeInput` / `buildBadgeContext` 사용 위치(트리·표·노드 블록) 요지.  
   - `plannodeTreeV1` / `pilotBridge` / `projects`: `metadata` I/O.  
   - `plannodePilot.js`: `showEdit` 3트랙, `toMdLine`·`getTreeText`, `triggerAI` + `AI_INTENT_BY_TYPE`.

5. **파이프라인 다이어그램 (mermaid)**  
   - `NodeEdit` → `metadata.badges` → `buildBadgeContext` + 트리 MD / PRD / `buildPrompt` → (미래) `callAI` / 지금은 클립보드·`#ai-result`.  
   - 스토어 동기: `onPersist` → `pilotNodesToStore` (참고: [`pilotBridge.ts`](src/lib/pilot/pilotBridge.ts)).

6. **AI 탭 / `OutputIntent` 정합 (재검토 포인트 반영)**  
   - `triggerAI`는 [`AI_INTENT_BY_TYPE`](src/lib/pilot/plannodePilot.js) → `buildPrompt(..., outputIntent, 'root')` — **PRD / SCREEN_LIST / FUNCTIONAL_SPEC / IA_STRUCTURE** 사용.  
   - `promptMatrix`에 `WIREFRAME_SPEC` 텍스트는 있으나 **현 UI 네 분석 버튼으로는 `WIREFRAME_SPEC`이 선택되지 않음** (후속: 버튼/매핑 보완 시 문서에 업데이트).

7. **테스트·검증**  
   - `npm test` / [`badgePromptInjector.test.ts`](src/lib/ai/badgePromptInjector.test.ts) 범위(빈/단일/복합).  
   - `build` (선택 1문장).

8. **하네스·후속**  
   - [`NEXT7_SCOPE.md`](.cursor/harness/NEXT7_SCOPE.md), [`TASK.md`](.cursor/harness/TASK.md)의 NEXT-7(요지만).

9. **변경 시 체크리스트 (보완용)**  
   - `callAI` 연결 시: env, CORS, 키 비노출, `iaExporter` 스텁 제거.

   - `WIREFRAME_SPEC` UI 노출 시: `plannodePilot` + `+page.svelte` AI 버튼.
